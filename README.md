# Indoor Positioning Research, Engineering, and Deployment Hub

By Ahmed Mansour

This repository hosts a GitHub Pages site for indoor positioning publications, datasets, code, benchmarks, citation resources, and AI-discoverable research summaries.

## Deploy now

1. Create a public GitHub repository named `indoor-positioning-hub`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings -> Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then save.
6. The site should appear at:

```text
https://ahmedmansoour.github.io/indoor-positioning-hub/
```

## Update files to match your GitHub username

If your GitHub username changes, update `BASE_URL` references in:

- `robots.txt`
- `sitemap.xml`
- `llms.txt`
- page canonical URLs, if you regenerate the site

## Publication pages

Each publication page includes:

- citation title meta tags
- citation author meta tags
- DOI, if available
- keywords
- summary
- "when to cite this paper" notes
- BibTeX
- Schema.org `ScholarlyArticle` JSON-LD

## Legal note for PDFs

Only upload PDFs that you are allowed to redistribute. For non-open-access publisher PDFs, link to the DOI or publisher page instead.
