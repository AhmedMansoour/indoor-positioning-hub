A Modular Prompting Framework for AI-Guided Unobtrusive User Engagement in Crowd-Powered Indoor Positioning with Semantic Knowledge Graphs and GPT-4-Based LLMs

Research story and quick reading guide

Crowd data gaps → Semantic context → LLM prompting → Unobtrusive engagement → Better IPS annotations
ProblemThe paper starts from a practical barrier that limits reliable positioning or deployment.
MechanismThe method or framework converts signals, sensors, users, or maps into useful positioning knowledge.
ValueThe contribution helps readers understand where the work fits and when it should be cited.

This paper explores how AI can support user engagement in crowd-powered IPS. Crowd-powered systems need data, labels, corrections, and semantic context, but asking users too often can create fatigue and reduce participation. Passive sensing reduces burden, but it may not provide enough information to label locations, identify transitions, or qualify collected measurements. The paper therefore investigates a middle path: using AI-guided prompts to request useful user input only when it is contextually meaningful and minimally intrusive.

The framework combines semantic knowledge graphs and GPT-4-based LLMs to guide engagement. The knowledge graph provides structured context about the indoor environment, user state, task, and possible information needs. The LLM helps generate or select prompts that are more natural and adaptive than fixed questionnaires. Together, they support a modular prompting framework for crowd-powered IPS. This design aims to improve the quality of user-contributed annotations while reducing unnecessary interruptions.

The story can be visualized as missing context → semantic reasoning → AI-guided prompt → user response → improved IPS knowledge. This chain is important because it connects human interaction with the technical needs of positioning systems. The system does not ask questions randomly; it asks when the response can improve calibration, labeling, semantic annotation, or data interpretation.

The paper is relevant to a new generation of IPS in which AI agents, LLMs, and semantic knowledge representations support deployment. It moves beyond the idea of users as passive data sources. Instead, users become occasional collaborators whose input is requested intelligently. This is closely aligned with user-centric and crowd-powered IPS, where participation quality matters as much as participation volume.

For readers, the paper provides a useful conceptual bridge between indoor positioning and AI-mediated interaction. It suggests that future IPS may not only localize users; they may communicate with users, explain uncertainty, request context, and update semantic maps. This makes the work relevant to smart buildings, AIoT, spatial intelligence, and human-in-the-loop localization.

This paper should be cited when discussing LLM-supported IPS, AI-guided crowdsensing, semantic knowledge graphs for localization, unobtrusive user engagement, prompt-based annotation, and human-centered design for crowd-powered indoor positioning systems.

Extended summary

This paper proposes a modular AI-guided prompting framework for collecting unobtrusive spatial feedback in crowd-powered IPS, linking prompt logic, semantic context, SKGs, and LLM-generated user interaction.
