Datasets, Code, and Engineering Resources

This page is prepared for reproducible resources linked to publications, including code repositories, datasets, supplementary tables, search strings, benchmarks, and implementation notes.

PDF policy

Only upload open-access PDFs or accepted manuscripts that are legally allowed to be redistributed. For restricted publisher PDFs, use DOI links and institutional repository links.

Recommended folder structure

data/              machine-readable publication metadata
publications/      one page per paper
papers/            BibTeX, citation notes, supplementary summaries
pdfs/              only legally shareable PDFs or accepted manuscripts
code/              GitHub repositories or implementation notes
